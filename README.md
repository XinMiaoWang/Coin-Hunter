# Coin-Hunter

* 遊戲介紹

    總共有三個關卡，玩家每收集5個金幣即可前進下一關卡。初始生命為2條，之後每通過一關就會送一條生命，生命歸零即通關失敗。

* 操控方法

    方向鍵"左"和"右"控制左右移動，空白鍵跳躍。
